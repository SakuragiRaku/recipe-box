レシピボックス 仕様書


============================================================
概要
============================================================

アプリ名:   レシピボックス
ジャンル:   レシピ管理 + 材料逆引き + 買い物リスト

どんなアプリ?
  お気に入りのレシピを材料・手順付きで登録して管理するアプリ。
  「今日何作ろう？」をサポートする2つの検索モード搭載。
    料理 → 材料: レシピを選ぶと必要な材料と手順を確認
    材料 → 料理: 冷蔵庫にある材料を入力すると作れるレシピを逆引き
  今週作りたいレシピから買い物リストを自動生成する機能付き。

誰が使う?
  自炊する人、献立に迷う人、食材を無駄にしたくない人。


============================================================
機能一覧
============================================================

レシピ登録/編集
  料理名
  カテゴリ (主菜 / 副菜 / 汁物 / ごはん / 麺類 / サラダ / デザート / その他)
  調理時間 (5分 / 10分 / 15分 / 20分 / 30分 / 45分 / 60分 / 60分以上)
  難易度ピル (かんたん / ふつう / こだわり)
  材料リスト (名前 + 分量を複数行)
  作り方 (手順を番号付きで複数行)
  ひとことメモ (自由記述)
  レシピの削除

レシピ一覧 (メイン画面)
  カード形式で一覧表示 (カテゴリカラー + 料理名 + 調理時間 + 難易度)
  カテゴリ別タブフィルター (すべて + 8カテゴリ)
  料理名での検索
  カードタップでレシピ詳細表示
  空の時は登録誘導メッセージ

材料から探す (逆引き機能)
  材料入力エリア (タグ形式で複数の材料を入力)
  入力した材料で作れるレシピを検索
  一致度でソート (完全一致 > 部分一致)
  各レシピに「足りない材料」を表示
  一致率バッジ (100% / 80% / 60% など)

買い物リスト
  作りたいレシピにチェックを入れると材料を自動集計
  重複材料はまとめて表示
  買い物済みチェック機能
  リストクリアボタン

テーマ
  ダークモード / ライトモード切り替え


============================================================
画面構成
============================================================

ヘッダー
  ロゴ + アプリ名
  ナビゲーション (レシピ / 材料から探す / 買い物リスト)
  テーマ切替ボタン

レシピ一覧画面 (メイン)
  カテゴリタブ (すべて / 主菜 / 副菜 / ...)
  検索バー
  レシピカードグリッド
  右下にFABボタン (レシピ追加)

レシピ詳細画面
  料理名 + カテゴリバッジ + 調理時間 + 難易度
  材料リスト (チェックボックス付き)
  作り方 (番号付きステップ)
  ひとことメモ
  編集 / 削除ボタン
  買い物リストに追加ボタン

材料から探す画面
  材料入力エリア (Enterでタグ追加)
  検索ボタン
  検索結果 (一致率順にレシピカード表示)
  各カードに一致率バッジ + 足りない材料表示

買い物リスト画面
  レシピ選択チェックリスト
  自動集計された材料リスト
  買い物済みチェック
  リストクリア

レシピ登録/編集モーダル
  料理名 / カテゴリ (セレクト) / 調理時間 / 難易度
  材料入力 (名前 + 分量の動的行追加)
  手順入力 (テキストエリアの動的行追加)
  ひとことメモ
  保存 / キャンセル


============================================================
技術構成
============================================================

フレームワーク: なし (HTML / CSS / JavaScript)

ファイル構成

      仕様書.txt       この仕様書
    index.html          メインHTML

============================================================
LocalStorageキー
============================================================

recipe-box-recipes     レシピデータの配列
                       各レシピ: id, name, category, cookTime, difficulty,
                                 ingredients (配列: name, amount),
                                 steps (配列), memo, createdAt
recipe-box-shopping    買い物リスト (選択レシピIDの配列 + チェック状態)
recipe-box-theme       light / dark
