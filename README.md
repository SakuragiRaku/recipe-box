# 🍳 レシピボックス

レシピ管理 & 材料逆引きアプリ。

お気に入りのレシピを登録・管理し、手持ちの材料から作れる料理を逆引き検索。買い物リストの自動生成も搭載。

## 機能

- 📖 レシピ登録 (材料・手順・カテゴリ・難易度)
- 🔍 材料から探す (一致率順に逆引き検索)
- 🛒 買い物リスト自動集計
- 🏷 8カテゴリの色分けフィルター
- 🌙 ダークモード / ライトモード

## 使い方

1. `index.html` をブラウザで開く
2. ＋ ボタンからレシピを登録
3. 「材料から探す」で冷蔵庫の食材を入力して検索！

## 技術

HTML / CSS / JavaScript / LocalStorage
